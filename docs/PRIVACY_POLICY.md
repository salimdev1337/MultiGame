- **Game Statistics**: Scores, game completion times, number of moves, achievements unlocked
- **Usage Data**: Games played, session duration, feature usage patterns
- **Performance Data**: App crashes, errors, and performance metrics

## How We Use Your Information

We use the collected information for the following purposes:

1. **Game Functionality**: To save your game progress, scores, and achievements
2. **Leaderboards**: To display global rankings and compare scores with other players
3. **App Improvement**: To analyze usage patterns and improve game features
4. **Performance Monitoring**: To identify and fix bugs, crashes, and performance issues
5. **Analytics**: To understand which games are most popular and how users interact with the App
